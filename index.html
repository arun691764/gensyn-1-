    Welcome to Compute Quest

    This mini-quiz teaches core ideas behind the Gensyn protocol (decentralised ML compute). Answer questions to earn score and badges — these show you're a community contributor.

    How to play
    1. Choose a compute node (affects rewards).
    2. Click Start Task to get a question based on Gensyn docs.
    3. Select the correct option(s) and wait for evaluation.
    4. Save your score to the leaderboard and share with the community.
    Why this helps

    Building and sharing demos like this increases community engagement. Tag the project on social channels when sharing demo results.
